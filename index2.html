01 — What is ExhiBytes
16file formats accepted
1×click to refresh all Word citations
∞exhibits per project
0cloud uploads — fully offline

Your exhibit bundle,
under control.

ExhiBytes is a Windows desktop application built for lawyers, litigators, and legal professionals who deal with exhibit bundles. Whether you are preparing for trial, an arbitration, or a transactional filing, ExhiBytes handles the tedious parts.

Drop in your documents — PDFs, Word files, Excel sheets, emails, images, and more — and ExhiBytes automatically converts, numbers, and organises them into a sequentially labelled exhibit list. Change the order and the labels update instantly.

The core feature is Word integration: insert live citation controls directly into your submissions, then refresh them all at once whenever your exhibit list changes. No more hunting through a hundred-page document by hand.

100% offline — your documents never leave your machine

04 — System Requirements

What you need
to run it.

ExhiBytes is a lean desktop app. The only heavyweight dependency is LibreOffice — and only if you need to convert Office files.

OSWindows 10 (64-bit) or Windows 11. macOS and Linux are not supported in this release.
RAM4 GB minimum. 8 GB recommended for smooth Office file conversion.
Disk500 MB for the application, plus space for your exhibit files.
Display1280 × 720 or higher.
LibreOfficeVersion 7 or later. Required only for converting DOCX, XLSX, PPTX and other Office formats to PDF. The installer will offer to install this automatically.
MS WordRequired only for the citation feature. Office 2013 or later with COM automation support.
InternetNot required after installation. ExhiBytes is fully offline.
PDF-only users If you only work with PDF files, LibreOffice is not required. It is only needed to convert Office format files. The app will display a warning if you add a non-PDF file without LibreOffice installed.

Installation Instructions

  1. Download the ZIP file using the button on the left and save it locally.
  2. Extract the ZIP file. Right-click and select Extract All…
  3. Right-click Install.bat and select Run as administrator. The installer checks for LibreOffice and offers to install it if missing.
  4. Follow the prompts. ExhiBytes is copied to C:\Program Files\ExhiBytes\ with Desktop and Start Menu shortcuts.
  5. Launch ExhiBytes from the Desktop shortcut. No further configuration required.
⚠ LibreOffice — Required to convert Office documents (DOCX, XLSX, PPTX) to PDF. The installer detects whether LibreOffice is present and will offer to run the bundled LibreOffice_26.2.2_Win_x86-64.msi if not. Skip if you only work with PDFs.
